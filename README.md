## For development in local

### Pre-requirements

- node 14+

### Setup

- install npm package

```bash
npm install
```

- start the app

```bash
npm run graphqlserver
```

- The app could be visited at http://localhost:3000
- Demo Graphgql could be visited at http://localhost:3002
